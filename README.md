# Strive - Premium Daily Habit & Goal Tracker PWA

**Strive** is a high-fidelity, glassmorphic Progressive Web App (PWA) designed to help users establish, track, and maintain daily habits. Built using Vanilla HTML5, CSS3, and JavaScript, the app is fully optimized for mobile and desktop screens and functions 100% offline.

👉 **Live App Website**: [https://harisasi006.github.io/habit-tracker/](https://harisasi006.github.io/habit-tracker/)

---

## 🌟 Key Features

1. **Circular Progress Ring & Weekly Summary**: Displays your daily completion percentage in real-time alongside a visual indicator for the last 7 calendar days.
2. **Multi-Theme Engine**: Toggle seamlessly between three premium, highly responsive color palettes:
   *   **Midnight Slate** (Default deep slate with blue/purple accents)
   *   **Cyberpunk Neon** (Dark theme with glowing orange/yellow accents)
   *   **Emerald Forest** (Dark theme with glowing mint/lime green accents)
3. **Web Audio Synthesizer**: Generates a clean, glassy arpeggiated chord sound effect when checking off a habit. Runs fully offline using the browser's native `AudioContext` (no heavy audio asset downloads required).
4. **Canvas Confetti System**: Triggers a physics-based particle confetti explosion that rains down on the screen when you complete all active habits (100% progress).
5. **Interactive Performance Analytics Dashboard**:
   *   **Total Completions**: Tracks total historical check-ins.
   *   **Best Streak**: Records your longest consecutive day streak.
   *   **Success Rate**: Computes your average consistency percentage over the last 30 days.
   *   **Individual Progress Bars**: Shows progress breakdown for each habit.
6. **GitHub-Style Contribution Map**: A grid of blocks showing the last 30 days, lighting up and glowing based on daily completion levels (0% to 100%).
7. **Category Filtering**: Organize habits by tags (**Work**, **Health**, **Mind**, **Fitness**, **Personal**) and filter them on your dashboard using custom navigation tabs.
8. **Weekly Targets**: Set specific target days per week for each habit (e.g., Gym 3 times a week, Drink Water 7 times a week) and track weekly progress bars inside each habit card.
9. **JSON Data Backup (Sync)**: Easily export all habits and tracking history into a local JSON backup file or restore it to sync your data.
10. **Habit Customization Modal**: Add or edit habit names, categories, weekly goals, emoji icons, and custom accent colors in place.

## 📲 Installation

Since Strive is a **Progressive Web App (PWA)**, you can install it directly onto your phone's home screen or laptop's desktop:

### On Mobile (Android/iOS)
1. Open the live link in **Google Chrome** (Android) or **Safari** (iOS).
2. Tap the browser's menu button (three dots in Chrome, or the Share button in Safari).
3. Select **"Install App"** or **"Add to Home Screen"**.
4. A custom **Strive** app icon will appear on your mobile home screen.

### On Desktop (Windows/Mac)
1. Open the live link in Google Chrome or Microsoft Edge.
2. Click the **Install** icon in the right side of the URL address bar.
3. The app will open in its own standalone window and place a shortcut on your Desktop.
